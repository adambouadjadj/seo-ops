Identifier les pages à optimiser

Filtre les requêtes avec impressions élevées + position 5–10 + CTR < 1.5%. Ignore les FP (.html) — gérées par la prod.

Checklist par page

  1. Le title contient-il la requête exacte ou une variante proche ?
  2. La H1 / intro adresse-t-elle directement l'intent de la requête ?
  3. La meta description est-elle incitative (si page contrôlée) ?
  4. La page traite-t-elle le sujet en profondeur ou c'est une liste générique ?

Ex : pos 9, 2 000 impr., CTR 0.5% → title à retravailler + vérifier que la requête est dans la H1. Pos 2, CTR 3.5% → ne pas toucher.
